## 运行

# 启用 pnpm
npm install -g pnpm

# 安装依赖
pnpm install

# 更新依赖
pnpm run update

# 运行
pnpm run dev

# 编译打包
pnpm run build

# 更新同步
pnpm run sync

# 发布到 GitHub Pages
pnpm run deploy

```
